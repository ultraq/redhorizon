
Red Horizon Explorer 🔎
======================

A file and media explorer application for viewing classic Command & Conquer game
assets.


Usage
-----

Requires Java 25 on either macOS 12 Monterey and newer or Windows 10 64-bit and
newer.

1. Either open the project in an IDE and run the main method of the `Explorer`
   class, or from a terminal you can run `./gradlew :redhorizon-explorer:run`.
